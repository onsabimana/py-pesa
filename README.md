# py-pesa
A python implementation of M-PESA API. M-PESA is the brand of the most popular mobile money service in Kenya and Africa owned and operated by SAFARICOM. Various mobile money services exist across Africa under different names such as MTN Mobile Money, TIGO cash...
SAFARICOM has not yet provided client API for the M-PESA service.This project is motivated by an earlier implementation of the API by PlusPeople https://github.com/pluspeople/pesaPi.
